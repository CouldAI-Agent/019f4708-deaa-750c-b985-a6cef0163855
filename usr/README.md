# GameVault

GameVault is a cross-platform application to explore and discover video games. It features a modern, responsive layout that adapts to mobile, tablet, and desktop screens, presenting games in an attractive grid view with a dark gaming aesthetic.

## Features
- **Responsive Grid:** Adapts seamlessly across phone, tablet, and desktop widths.
- **Game Library:** Browse an extensive mocked directory of trending games.
- **Detailed Views:** Rich detail screens displaying game cover art, ratings, genres, and descriptions.
- **Cross-Platform:** Built with Flutter, fully supporting Android, iOS, Web, and Desktop.

## Tech Stack
- Flutter
- Dart
- Material Design 3

## Getting Started
To run the app locally:
```bash
flutter pub get
flutter run
```
